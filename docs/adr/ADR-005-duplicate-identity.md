# ADR-005: Duplicate Identity / Merge Governance

**Status:** Accepted | **Workstreams:** P0, WS-4

## Decision

One athlete = one canonical profile. Merges require explicit per-group approval (G3-M). `Game`/`GameStat` evidence is reassigned to canonical identity, never rewritten. Merges complete before lifetime accumulation (G3 before G4).
