# ADR-009: Carryover as Discounted Prior

**Status:** Accepted | **Workstreams:** WS-6, B1

## Decision

At June rollover, players receive a **discounted prior** from the prior age-group rating. This is distinct from game-count Bayesian shrinkage. Carryover parameters are WS-3-owned; carryover weight is explicit. Carryover **never satisfies eligibility threshold alone**.
