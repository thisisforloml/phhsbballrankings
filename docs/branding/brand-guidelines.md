# Brand Guidelines

## Brand name

**Peach Basket** — verified Philippine youth basketball rankings and player visibility.

## Visual identity (application UI)

The application UI uses a **court navy + hardwood orange** palette established before the Peach Basket rebrand. Phase 1 documents and tokenizes this system — it does not replace it.

| Role | Usage |
| --- | --- |
| **Primary (navy)** | Headers, nav, primary buttons, structural surfaces |
| **Accent (orange)** | CTAs, active states, stat emphasis, kickers |
| **Neutral gray** | Backgrounds, borders, body text hierarchy |
| **Semantic** | Success / warning / danger for state only |

## Peach color (brand only)

Peach (`#F4A261`) belongs to:

- Logo and wordmark
- Marketing materials
- Social assets

Do **not** use peach as the primary application UI color.

## Logo

SVG is the default format. PNG remains as fallback for email and legacy embeds.

See [Logo Usage](./logo-usage.md).

## Typography

- **Geist** — all prose, headings, navigation, labels
- **Geist Mono** — numeric data only

See [Typography](./typography.md).

## Icons

- **Lucide React** — general UI (search, menu, lock, stars)
- **Sports Icons** (`@/components/icons/sports`) — basketball-specific

See [Icons](./icons.md).
